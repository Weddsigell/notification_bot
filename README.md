# Чат бот для получения уведомлений по работам на Devman
С помощью бота вы можете не переживать, что забудете о работе на проверке

## Как установить
Склонируйте данный репозиторий к себе
```bash
git clone https://github.com/Weddsigell/notification_bot.git
cd notification_bot
```

## Переменные окружения
Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл .env в корне проекта и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.

Доступны 3 переменныx:
* TG_BOT_TOKEN - секретный токен вашего телеграмм бота, который можно получить [тут](https://telegram.me/BotFather).
* DEVMAN_TOKEN — секретный токен авторизации для работы с api devman, можно получить [тут](https://dvmn.org/api/docs/)
* TG_CHAT_ID — ваш телеграмм id для получения уведомлений именно вами, можно получить [тут](https://telegram.me/userinfobot)

## Как запустить
Для запуска сайта вам понадобится UV. Как установить вы найдете [тут](https://github.com/astral-sh/uv).

* Введите команду 
    ```bash
    uv run main.py
    ```
* Заходите в вашего бота и запускайте его командой
    ```
    /start
    ```
