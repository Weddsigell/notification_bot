# Чат бот для получения уведомлений по работам на Devman

1. Скачайте репозиторий
2. Установите UV
3. Создайте .env файл
4. Добавьте в .env 3 переменных окружения
    1. TG_BOT_TOKEN - берется в телеграмме у @BotFather
    2. DEVMAN_TOKEN - находится по [адресу](https://dvmn.org/api/docs/)
    3. TG_CHAT_ID - берется в телеграмме у @userinfobot
5. Введите команду 
    ```
    uv run main.py
6. Заходите в вашего бота и запускайте его командой
    ```
    /start
